# Mouse embryo E9.5 Stereo-seq dataset

Download the dataset from https://ftp.cngb.org/pub/SciRAID/stomics/STDS0000058/stomics/mouse_E9.5_embryo.h5ad
Then try to explore the dataset, and figure out some interesting questions and answers to the questions by analyzing the dataset.
This is a 3D Stereo-seq dataset, you should reconstruct the 3D embryo structure then perform the analysis on the 3D embryo structure.
