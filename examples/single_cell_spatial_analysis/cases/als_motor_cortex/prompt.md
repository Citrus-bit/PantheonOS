# ALS motor cortex and spinal cord single-nucleus multiome dataset

Download the dataset from https://datasets.cellxgene.cziscience.com/0ab54d91-066c-4223-a9ea-6a3b0d1adef4.h5ad
Then try to explore the dataset, and figure out some interesting questions and answers to the questions by analyzing the dataset.
This dataset is a processed snRNA-seq and snATAC-seq data for ALS and controls, with cell-type annotations.
Publication: https://doi.org/10.1093/brain/awaf426
