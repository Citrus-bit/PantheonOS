# Agent skills for analyzing single-cell/Spatial Omics data

Here are some skills about the best practices for analyzing the single-cell/Spatial Omics data.
When you need to do the specific analysis tasks described in this file,
you can load the related skill files linked here to help you.

## Map single-cell data to spatial data

If you have both single-cell and spatial data for same/similar sample,
you should try to mapping the single-cell data to the spatial data,
so that you can impute unobserved genes and enhance the spatial data.

When you need do this analysis task, please load and read: `single_cell_spatial_mapping.md`.

## Visualize the 3D spatial data

If you have 3D spatial data, you can visualize the 3D spatial data to help you understand the data.
When you need do this analysis task, please load and read: `visualize_3D_spatial_data.md`.
